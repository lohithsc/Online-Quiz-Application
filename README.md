# Online-Quiz-Application
A web-based online quiz application developed using HTML, CSS, and JavaScript.
## Project Overview

The Online Quiz Application is a simple web-based quiz system developed using HTML, CSS, and JavaScript. The application presents multiple-choice questions to users, evaluates their answers, calculates the score, and displays the final result along with the correct answers for incorrectly answered questions.

This project demonstrates the use of front-end web technologies and JavaScript DOM manipulation to create an interactive quiz experience.

---

## Features

* Multiple-choice quiz questions
* Randomized option order for each question
* Automatic score calculation
* Result display after quiz completion
* Review of incorrect answers with correct answers
* Retry quiz functionality
* User-friendly interface

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## File Description

### index.html

Contains the structure of the quiz application and links all required CSS and JavaScript files.

### style.css

Provides styling for the quiz interface, buttons, questions, and result display.

### questions.js

Stores all quiz questions, answer options, and correct answers.

### quiz.js

Contains functions responsible for displaying questions and shuffling answer options.

### script.js

Contains the main application logic including answer validation, score calculation, result display, retry functionality, and answer review.

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Start answering the quiz questions.
5. View your score after completing the quiz.
6. Click **Retry** to attempt the quiz again.
7. Click **Show Answers** to review incorrect responses.

---

## Sample Functionalities

### Quiz Attempt

* User selects an answer for each question.
* Application checks whether the selected answer is correct.
* Score is updated accordingly.

### Result Generation

* Final score is displayed after all questions are answered.
* Incorrect answers are stored for review.

### Answer Review

* Displays:

  * Question
  * User's Answer
  * Correct Answer
